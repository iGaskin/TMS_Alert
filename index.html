About TMS Alerts

The Topical Memory System (TMS) is a system developed by The Navigators to help with the memorization of key verses that point to basic truths. This web application aims to help verse memory with weekly alerts tailored to your current pack of study. This application is open source, and can be used for any purpose, personal or commercial.

Simply enter your phone number, and prefered Verse Pack below to get started
